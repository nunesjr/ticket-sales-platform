# 🎟️ Event Ticketing Platform with Promoter System

## 🚀 Overview
A complete event ticketing platform designed to manage events, online ticket sales, and promoter-driven distribution.

This system enables event organizers to create, manage, and sell tickets while leveraging a built-in promoter system to increase reach and sales through referral-based strategies.

Built for real-world use, the platform centralizes event management, customer data, and sales performance in a single environment.

---

## 🔐 Code Availability
This project is private due to commercial use and client confidentiality.  
The source code is not publicly available.

---

## 🛠️ Technologies
- Backend: PHP / Node.js  
- Frontend: HTML, CSS, JavaScript  
- Database: Relational database (MySQL / Firebird)  
- Integrations: Payment systems, SMTP (email notifications)  
- Environment: cPanel hosting  

---

## 🔥 Core Features
- Event creation and management system  
- Online ticket sales with checkout flow  
- Customer registration and purchase tracking  
- Administrative dashboard for full control  
- Real-time sales monitoring  

---

## 💎 Advanced Features (Key Differentiator)
- Promoter-based sales system with custom referral links  
- Ranking system for top-performing promoters  
- Gamification with sales goals and incentives  
- Custom discount coupons and tracking  
- Performance analytics per promoter
